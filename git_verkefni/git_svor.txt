Git spurningar og svör

1. spurning. Hvað gera eftirfarandi Linux skipanir?
cd - Skiptir úr núverandi directory.
cd ~ - Sýnir home directory-ið.
ls - Listar upplýsingar um skrár í núverandi directory.
pwd - Finnur pathið á núverandi directory.
mkdir - Býr til directory.

2. spurning. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
git clone - Afritar git repository sem er nú þegar til. Tekur nánast allt data af afritinu sem server-inn hefur, 
allar skrár projectsins eru pullaðar. Þetta er gagnlegt vegna þess að það minnkar líkur á tapi skráa. 
git log - Sýnir hvað þú ert búinn að commita. Gagnlegt af því getur haft yfirlit yfir hvað þú ert að gera.
git status - Sýnir stöðu mála í working tree-inu. Gagnlegt af því það sýnir þér.
git diff -Sýnir breytingar á milli commita og working trésins. Gagnlegt að sjá breytingar sem þú hefur gert. 
git checkout - Skiptir um greinar eða endurkallar working tree skrár. 

3. spurning. Hver er munurinn á eftirfarandi git skipunum:
a) git diff - Sýnir breytingar á milli commita og working trésins.
b) git diff --staged - Sýnir breytingar sem hafa verið cache-aðar en ekki commitaðar. 
c) git diff commit1 commit2 - Sýnir breytingar á milli commit 1, og commit 2.

4. spurning. Hvað er útgáfustýring (e.version control)? 
Umsjón yfir breytingum skjala, forrita, vefsíða og önnur collections yfir upplýsingar.

5. spurning. Hverjir eru helstu kostir við að nota GIT?
Gerir þér kleift að búa til greinar til að prófa kóða með. Gerir þér kleift að vista allar útgáfur af vefsíðunni þinni. 
Þú getur séð hvað þú og eða samstarfsmaður þinn er búinn að vera að gera nýlega.

6. spurning. Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
Eins oft og mögulegt er, því meiri upplýsingar eru, því betra, í þessu máli þar að segja.

7. spurning. Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
Working directory er directory með upprunalegu file-um þínum undir 'stjórn' git.
Staging area er skrá sem geymir upplýsingar um hvað mun fara næst í næsta commiti sem þú gerir.
Repository er skrá eða staður þar sem saga vinnu þinnar er geymt.

8. spurning. Hvenær er sniðugt að nota greinar (branches)? 
Til dæmis þegar einhver önnur deild í fyrirtækinu sem þú vinnur hjá þurfa að þínum skrám á að halda,
þá getur þú notað branch til þess að búa til önnur afrit af því.
